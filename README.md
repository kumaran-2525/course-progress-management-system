# Course Progress Management System

A web-based Course Progress Management System developed using HTML5, CSS3, JavaScript, Supabase, and PostgreSQL.

The system helps manage student enrollment, lesson tracking, course completion progress, and faculty monitoring using role-based dashboards.

## 📌 Features

- Student & Faculty Login System
- Course Enrollment Management
- Lesson Tracking
- Progress Monitoring
- Faculty Dashboard
- Responsive UI Design
- Supabase Database Integration
- Role-Based Access System

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Supabase
- PostgreSQL

## 🗄️ Concepts Used

- DBMS
- ER Diagram Design
- Primary & Foreign Key Relationships
- Enrollment-Based Tracking
